## <a id="_____122"></a><a id="_135"></a>5\.4\. DataVault与BitMap技术

### <a id="_____124"></a><a id="_173"></a>Data Vault模型定义：按照Dan Linstedt的定义，Data Vault模型是面向细节的、可追踪历史的、一组有链连接关系的规范化的表的集合。它综合了三范式建模和星型模型的优点，其设计理念是满足企业对数据模型灵活性、可扩展性、一致性和对需求的适应性要求，是专门针对企业级数据仓库需要的一套建模方法

### <a id="_____126"></a><a id="_174"></a>Hub组件，是一个数据表，用于记录在业务应用中常用到的业务实体键值，如员工ID，发票号

### <a id="_____128"></a><a id="_175"></a>Link组件，通过存储相关业务实体间Hub表的SK（Surrorgate Key），以记录一对多，多对多的业务实体间关系，如员工与企业的雇佣关系，账户与客户之前的关系等

### <a id="_____130"></a><a id="_176"></a>BitMap可以理解为通过一个bit数组来存储特定数据的一种数据结构：由于bit是数据的最小单位，所以这种数据结构往往是非常节省存储空间的

## <a id="_____132"></a><a id="_136"></a>5\.5\. 知识图谱建模

### <a id="_____134"></a><a id="_177"></a>BEAF原则：业务原则（实体是以业务为中心的）、分析原则（任何一个实体都是为关系分析而服务的）、冗余原则（避免存放任何重复的信息）、效率原则（尽量轻便，只存储关键信息）

## <a id="_____152"></a><a id="_139"></a>6\.2\. 画像建模与应用

### <a id="_____154"></a><a id="_164"></a>定性分析：定性就是用文字语言进行相关描述。定量分析：定量就是用数学语言进行描述。定性分析与定量分析应该是统一的，相互补充的。

### <a id="_____156"></a><a id="_165"></a>用户画像常用来作为精准营销、推荐系统的基础性工作。其作用总体包括：精准营销、用户分析、数据挖掘、服务产品、行业报告&用户研究
